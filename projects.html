Note

Note: all projects source code will be there as some repositories are private for academic integrity purposes.

  1. Marvel Movie Success Predictor

    In this project I worked with a team of 5 other people where we tried to discover if there was a way to predict the success of a marvel movie, both critically and commercially, based on general movie statistics like its run time, budget, year released, etc. This was mainly done with python inside of Jupyter Notebooks where we used libraries like pandas, numpy, patsy, statsmodels, etc. Through this project we found that things like production budget did have a positive correlation with the success of the movie.

  2. Top Spotify Songs

    This was a project I worked on last Summer where I attempted to make a Spotify web application that would allow a user to log into their spotify account and then show what their top songs are for all time on the platform. It would also allow you to view main profile features like number of playlists or number of friends on the app.

  3. MineSweeper

    A fun little pet project which recreates MineSweeper fully using java code. It allows you to play the game from within the computer's terminal. Doesn't do too much other than that, but can be a fun little time killer.
